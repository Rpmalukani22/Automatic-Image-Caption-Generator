This is the most earliest phase of our project. We began our project by having a look on various domains followed by topic selection and concept understanding.

Exploring the domains

We started our exciting journey by exploring various domains of Computer Science. Finally, we selected Computer Vision and Natural Language Processing as our project domain.

Selection of the topic

We decided to go with today's the most technology i.e Deep Learning hence we explored many ideas for the same. Finally we decided to create an Automatic Image Caption Generator using the state of the art methods.

Basic concept understanding

We started learning the underlying concepts of Machine Learning, Deep Learning using various learning platforms like Coursera and YouTube.

In this phase we completed the literature survey on various papers related to Automatic Image Captioning.

Exporing the methods used in various papers

We explored different methods used in various papers for achieving the solution of generating captions for the images.

Understanding the underlying technologies and tools

We read about various models, tools and technologies used by different papers. Also we explored various datasets used by them.

Survey paper and Poster writing

We presented survey paper and poster on "Automatic image captioning methods" in International Conference on Recent Advances in Computer Science and Technology (ICRACST- 2020).

Have a look

In this phase we started implementing our project. We have used CNN as a feature extractor and RNN as a language model.

Data pre-processing

We explored various datasets such as Flikr-8K, Flikr-30K and MSCOCO for our task. Finally we decided to go with MSCOCO dataset.

Model selection and training

We selected Encoder-Decoder model for our project. We have used CNN as Encoder for extracting features from the image and RNN as language translator for caption generation. We trained our model on MSCOCO dataset over approximately 82K images.

Testing and Deployment

Finally we tested and deployed our trained model using Flask framework for backend and Angular framework for frontend web development.
